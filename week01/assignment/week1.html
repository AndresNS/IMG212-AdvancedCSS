IMG 212 Assignment 1

Produce an HTML page similar to the one we created in class. Please use standards compliant HTML5 that includes the following:

  • A 'header' area with an 'h1' element that includes the title of your assignment (i.e. IMG 212 Assignment 1).
  • Two 'p' elements with placeholder text.
  • A 'ul' or 'ol' element that includes at least three li elements.
  • At least two 'a' elements somewhere in the page.

Now develop the CSS:

Use the 'style' element in your HTML document 'head' to add the following rules:

  • A :root rule to set the font size to 14px.
  • A p rule that uses rems to set font-size (any size you like, as long as it is visible).
  • A ul rule that adds a border (any type) around the 'ul' element.
  • An li rule that forces the border to inherit to all 'li' elements.
  • A class that modifies margin-bottom spacing between your choice of elements using ems.
  • Add four additional rulesets (and modify your HTML, if necessary) to produce selectors of each of the following specificities:
    • 1,0,0
    • 0,2,0
    • 0,0,2
    • 0,1,1
  • Create a custom property of your choice and apply it within at least two rulesets (use listings 2.27, 2.28 and 2.29).
